Privacy Policy

Overview

This Privacy Policy explains what data is collected when you use the NearHermitCraft Minecraft server network and related services, how that data is used, how it is stored, and what rights you have.

We only collect data that is needed to run, secure, and improve the server. This includes basic connection information and gameplay statistics that help admins understand server health and player activity.

Some features linked from this site are run by third parties and may collect data under their own privacy policies.

NearHermitCraft will never sell your data. All data is stored locally in a pseudonymised format and is only shared when required for functionality, anonymised, or with appropriate consent.

Our services are not intended for children under the age of 18. By using NearHermitCraft and its related services, you agree to this Privacy Policy.


Cookies

When you visit the Map, the web-application uses cookies to save the configurations of a user.


Data Collection

When connecting to NearHermitCraft's Minecraft server(s), the player's IP address is logged and stored locally on the server.
If these logs were to be shared, mclo.gs would ensure IP addresses are hidden.

For more information about what data mclo.gs collects, see their Privacy Policy.


Player Analytics data is gathered and stored locally on the server and is not transferred to third parties.
Transfer of this data to a third party will require consent of all parties involved, unless the pseudonymised user data is removed/anonymized before the transfer.

If there is a data breach the owner of the server will notify affected users.

How is this data used:

This data is used for providing NearHermitCraft Admins with Analytics about their server, so that they can make more educated decisions when making changes to their server.

Information calculated from the available data:

  • How "healthy" the server is
  • Are players having people to play with
  • Are players staying on the server
  • How many new players stay as regular players
  • How many unique players stay on the server each day
  • What is the most played world, and in networks what server is the most played server

Data we gather (Linked to UUID):

  • Your UUID (Unique ID) and Player name
  • Approximate country derived from IP address
  • Used display names (Such as those changed with /nick)
  • Date you first joined the server
  • Sessions
  • Join and Leave dates when you have been on the server to calculate Playtime
  • the IP/address that you joined the server with (eg. play.server.com)
  • Time information about time spent in each World and GameMode
  • Time information about being AFK
  • How many mobs were killed
  • How many deaths occurred
  • Ping information (delay to the server)
  • Player kills (Linked to user), time and weapon item name

Data we gather (Anonymous):

  • Players Online every minute
  • How many times each command is used
  • How many entities and chunks are loaded on the server

How is this data stored:

The data is stored in a SQL Database in a pseudonymized format. Linking the data to a real person requires a key - in this case UUID and access to Mojang's Account databases (These databases are not publicly available).

Your rights:

  • Right to request the removal of your data
  • Right to be informed of what data is held of you (Access to own player page on default Plan setup)
  • Right for rectification

To exercise these rights, please contact us via the Contact Us page, or contact an Admin on Discord.
